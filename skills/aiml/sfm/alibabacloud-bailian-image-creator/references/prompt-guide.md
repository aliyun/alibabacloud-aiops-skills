# Professional Prompt Design Guide

The key to generating high-quality images is crafting artist-level prompts. Below are the core elements of professional prompts:

## Prompt Structure Formula

```
[Subject description] + [Scene/environment] + [Lighting/mood] + [Composition/angle] + [Art style] + [Quality parameters]
```

## 1. Lighting Design

| Lighting Type | Keywords | Effect Description |
|--------------|----------|-------------------|
| Natural light | `golden hour light`, `soft daylight`, `overcast diffused light` | Soft, realistic natural quality |
| Artificial light | `studio lighting`, `ring light`, `neon glow` | Controllable, precise lighting effects |
| Dramatic light | `chiaroscuro`, `Rembrandt lighting`, `rim light`, `backlit silhouette` | Strong contrast, emotional tension |
| Ambient light | `volumetric lighting`, `god rays`, `hazy atmosphere` | Spatial depth, mystery |
| Color light | `warm tones`, `cool blue cast`, `cinematic color grading` | Mood and tone control |

**Example**:
```
Soft golden hour light streams through the window at an angle, creating a warm Tyndall effect, Rembrandt-style triangle lighting on the subject's face, background in soft shadows
```

## 2. Scene Construction

| Scene Layer | Description Points |
|------------|-------------------|
| Foreground | Adds depth of field, guides the eye (e.g., blurred flowers, props) |
| Midground | Where the subject is located, richest in detail |
| Background | Environmental context, mood setting (e.g., architecture, sky, distant mountains) |
| Environmental details | Materials, textures, weather, time of day, season |

**Example**:
```
Foreground of blurred cherry blossom petals falling, midground of a woman in kimono standing on a stone bridge, background of a misty Kyoto ancient temple, distant mountains barely visible, light rain, wet ground with reflections, water vapor in the air
```

## 3. Character Description

| Dimension | Description Points |
|----------|-------------------|
| Appearance | Age, skin tone, hairstyle/color, facial features |
| Clothing | Style, material, color, accessories, texture details |
| Posture/Action | Standing/sitting pose, hand gestures, head angle, center of gravity |
| Expression | Eye direction, emotional state, micro-expressions |
| Environmental interaction | Touching, looking, walking, and other action relationships |

**Example**:
```
A 25-year-old East Asian woman with long black hair, slightly wavy over her shoulders, wearing a cream-colored linen dress, simple gold earrings, standing sideways, left hand gently touching the curtain, gazing out the window, looking thoughtful with a faint smile
```

## 4. Composition Principles

| Composition Type | Keywords | Best For |
|-----------------|----------|----------|
| Rule of thirds | `rule of thirds`, `off-center subject` | Portraits, landscapes, balanced and stable |
| Center symmetry | `centered composition`, `symmetrical` | Architecture, portraits, formal and dignified |
| Leading lines | `leading lines`, `diagonal composition` | Streets, corridors, depth perception |
| Frame composition | `framed composition`, `natural frame` | Doorways, archways, focusing on subject |
| Negative space | `negative space`, `minimalist composition` | Minimalist style, artistic expression |

**Angle Keywords**:
- `eye level` - Eye-level, friendly and natural
- `low angle` - Looking up, grand and imposing
- `high angle / bird's eye view` - Looking down, overview
- `dutch angle` - Tilted, uneasy and dynamic
- `close-up` - Close-up, detail and emotion
- `wide shot` - Full view, environmental context

## 5. Art Style Reference

| Style Category | Example Keywords |
|---------------|-----------------|
| Photography | `photorealistic`, `35mm film`, `DSLR`, `portrait photography`, `street photography` |
| Painting | `oil painting`, `watercolor`, `digital art`, `concept art`, `illustration` |
| Art movements | `impressionism`, `art nouveau`, `baroque`, `minimalism`, `surrealism` |
| Artist reference | `in the style of Studio Ghibli`, `Makoto Shinkai style`, `Greg Rutkowski style` |
| Rendering | `3D render`, `Unreal Engine`, `octane render`, `ray tracing` |

## 6. Quality Parameters

**Positive quality keywords**:
```
high resolution, 8K, ultra detailed, sharp focus, professional, masterpiece, best quality, intricate details, aesthetic
```

**Negative prompt template** (for negative_prompt):
```
low quality, blurry, pixelated, oversaturated, deformed hands, extra fingers, distorted face, uncanny valley, text, watermark, logo, signature, cropped, out of frame, worst quality
```
